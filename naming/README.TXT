readme for naming.awk
93-08-04

** naming.awk とは

姓名判断に基づいてよりよい名前を考案するプログラム。
貧弱な知識からよりよい名前を探してくれる。

** naming.awk の使い方

naming.awk、point.tab､kakusu.dicを同じディレクトリに置き、
そのディレクトリに移動する。
そこで次のようにコマンドラインから打ち込む。

	jgawk ｰf naming.awk <姓> <名>

姓と名の間はスペースを空ける。
後は、指示に従ってやればよりよい名前を示してくれる。
よりよい名前を作り出せない場合は、
その旨メッセージを表示して終了する。

** 例

「志村拓」の人運をチューンナップする。

------------
A>jgawk -f naming.awk 志村　拓
「志」の画数を教えてください: 
------------
「志」の文字の画数を知らないとこのように聞いてくるので、
画数を入力する。

------------
「志」の画数を教えてください: 7

あなたの名前は　志　村　拓
　　　　画数は　 7　 7　 8
1.天運: 14画 80点      2.地運:  8画 80点
3.人運: 15画 10点      4.外運: 15画 10点      5.総運: 22画 80点

どの運勢をチューンしますか？(1-5): 
------------
と、どの運勢をあげるか聞いてくる。
そこであげたい運勢を1〜5の中から選んで入力する。
ここでは人運(=3)をあげてみる。すると

ｰｰｰｰｰｰｰｰｰｰｰｰ
どの運勢をチューンしますか？(1-5): 3

氏名の3番目を18画にチューンします
 
志村麿
志村曙
志村鞭
志村癖
志村難

チューンの成果は
1.天運: 14画 80点      2.地運: 18画 80点
3.人運: 25画100点      4.外運: 25画100点      5.総運: 32画100点
ｰｰｰｰｰｰｰｰｰｰｰｰ
というように、知っている漢字で画数があうものを選び出してくれる。

** 関連ファイル

readme		このファイル

(※)が付いているファイルは必須

** bugs

- 知っている文字が少ない(辞書による)。

- 名前の候補の選び方が適当(詳しくはscript参照)。

- 一つの運勢だけをチューンアップするので他の運が下がる場合が多い。

- そもそも姓名判断の精度が怪しい。
  数値だけで評価するものじゃないだろう...

**caution !!

- 画数辞書(kakusu.dic)のファイルにCTRL-Zがついている場合、それ以後漢字と
　その画数を入力しても辞書に付け加えられないので注意してもらいたい。

- ふつうDOS上のエディタで編集するとCTRL-Zがつけられることが多い。ついて
　しまったCTRL-Zをエディタ上で除去するには、MIFESの場合は、MIFES -ba と
　して起動した後、削除すればよい。VZの場合は、VZ -ez- として起動した後、
　削除する。
