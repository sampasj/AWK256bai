readme for hyou.awk
93-07-10 SASAKI Takesi

** hyou.awk とは

awkを使って簡単な表を作るだけのプログラムだ。
ASCIIの文字で表を作るほかに、
JIS罫線や 98独自の罫線にも対応している
(ただし俗にいう「半角罫線」には対応していない)。


** hyou.awk の使い方

syntax:	jgawk -f hyou.awk [-- option] [filename]

filename は原稿となるデータの入ったファイルである。
省略された場合、標準入力から読み込む。
option としては以下のようなものがある。

	-a	ASCII文字で表を作る(default)
	-j	JIS罫線で表を作る
	-n	PC98罫線で表を作る


** 原稿ファイルの書き方

単に、スペースやタブで各フィールドが分けられたテキストファイル。
各フィールドにつき、最大幅の枠が用意される。
普通のフィールドは、そこに左詰めで書かれる。
数字およびコンマからなるフィールド(正確には script を参照)は、
識別され自動的に右詰めで書かれる。
丸括弧で囲まれたフィールドはセンタリングされる。

空行があると、そこの境界線は太い線になる。


** bugs

- 何も要素がないデータを入力すると塵を吐く
	(Null in garbage out ^^;)。

- 丸括弧が両側に付いたデータをそのまま表示できない。
	(数値データの左詰め、なども同様)

- PC98罫線は標準ではないため、普通(?)の機種では見えない。

- JIS罫線はPC9801では見えない(JIS83に対応してないため)。
